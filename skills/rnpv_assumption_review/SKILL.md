# rNPV Assumption Review Skill

## Use When

Valuation or deal terms depend on assumptions that should be checked against diligence findings.

## Review Method

1. List core value drivers: eligible population, price, uptake, duration, share, POS, cost, timeline, royalties, milestones.
2. Tie each assumption to evidence, benchmark, expert judgment, or explicit uncertainty.
3. Identify assumptions affected by clinical, regulatory, CMC, IP, access, or competitive risks.
4. Build downside/base/upside scenarios and sensitivity drivers.
5. Identify breakpoints that change pursue, renegotiate, pause, or reject recommendation.

## Output Fields

- Assumption
- Current value or range
- Evidence basis
- Risk adjustment
- Sensitivity impact
- Diligence request

## Quality Boundary

Do not treat valuation output as precise when key assumptions are unsupported. Label unsupported assumptions and force scenario ranges.
