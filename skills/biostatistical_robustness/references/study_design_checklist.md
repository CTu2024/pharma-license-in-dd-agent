# Biostatistical Study Design Checklist

Use this reference when evaluating whether a clinical claim is statistically reliable.

## Required Documents

- Protocol and amendments.
- Statistical analysis plan and amendments.
- Randomization and blinding description.
- Sample size calculation and assumptions.
- Endpoint hierarchy and multiplicity plan.
- Estimand framework where applicable.
- Analysis population definitions.
- Missing data handling plan.
- Interim analysis and stopping rules.
- Tables, figures, listings, datasets, and sensitivity analyses.

## Design Checks

Capture:

- Study objective: exploratory, dose-finding, proof-of-concept, pivotal, confirmatory.
- Primary endpoint and timing.
- Secondary endpoints and hierarchy.
- Comparator and allocation ratio.
- Power, alpha, effect size, variance, dropout, and assumptions.
- Randomization strata and baseline balance.
- Blinding and bias control.
- Analysis sets: ITT, mITT, PP, safety.
- Handling of rescue therapy, intercurrent events, and discontinuation.

## Phase-Specific Emphasis

### Phase 1 / 1b

- Avoid over-weighting exploratory efficacy.
- Check dose escalation and safety stopping rules.
- Treat biomarker and target engagement as supportive, not confirmatory.

### Phase 2

- Check whether signal is prespecified or post hoc.
- Check multiplicity and subgroup inflation.
- Check whether effect size assumptions are credible for Phase 3.

### Phase 3

- Require strong control of Type I error.
- Require prespecified estimands, sensitivity analyses, and endpoint hierarchy.
- Scrutinize missing data, intercurrent events, and regional consistency.
