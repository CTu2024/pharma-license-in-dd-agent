# Evidence Traceability Skill

Use this skill for every Scouting and Primary Diligence output.

## Core Rules

- Separate facts, assumptions, interpretations, and recommendations.
- Every material claim needs a source or an explicit assumption label.
- Prefer primary source documents over summaries.
- Do not hide uncertainty. State when evidence is incomplete, indirect, outdated, or inconsistent.
- Record each source in the applicable memo or source register.

## Source Hierarchy

1. Regulatory correspondence, clinical study reports, protocols, statistical analysis plans, CMC batch records, audited quality documents.
2. Company data room summaries, investor decks, internal analyses.
3. Peer-reviewed publications, trial registries, conference materials.
4. Analyst reports, press releases, commercial databases, web sources.
5. Expert opinion and agent inference.

## Claim Labels

- `Fact`: Directly supported by cited source.
- `Assumption`: Needed for analysis but not proven by available sources.
- `Inference`: Reasoned interpretation from available evidence.
- `Open question`: Important point that cannot be resolved from current sources.

## Output Pattern

For each workstream, include:

- Key claims with source IDs.
- Unsupported or weakly supported claims.
- Source quality concerns.
- Documents needed to verify important claims.
